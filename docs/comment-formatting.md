# Форматирование comment в data.js

## Синтаксис

### Заголовки
```
## Заголовок 2 уровня
### Заголовок 3 уровня
```

### Горизонтальная линия
```
---
```

### Списки
```
- Пункт маркированного списка
* Альтернативный маркер

1. Нумерованный пункт
2. Второй пункт
```

### Инлайн-форматирование
```
**жирный текст**
*курсив*
`код`
```

### Ссылки
```
[текст ссылки](https://example.com)
```

## Пример

```json
{
  "comment": "## Описание проекта\n\nОсновные характеристики:\n- Пункт 1\n- Пункт 2\n\n### Детали\n\nДополнительная информация с [ссылкой](https://example.com).\n\n---\n\n*Курсивный текст* и **жирный текст**."
}
```
